### ESP32 Side (Acebott Board)

| Wire Color | ESP32 Pin | Function |
|------------|-----------|----------|
| Yellow     | SDA       | Data line (I2C) |
| Blue       | SCL       | Clock line (I2C) |

| Black      | GND       | Ground |
| Red        | 3V3       | Power |

### BME280 Sensor Side

| Wire Color | Sensor Pin | Function |
|------------|------------|----------|
| Blue       | SDA        | Data line |
| Yellow     | SCL        | Clock line |
| Black      | GND        | Ground |
| Red        | VIN        | Power input (matches 3V3 from ESP32) |
